YY-CHRでスーパーファミコンのDQ3のROMのプレイヤーキャラ画像を編集するために、あると便利なファイル

dq3.adf      … パターンファイル
dq3.setting  … PALパレットファイルの読み込み設定
DQ3.jumplist … ジャンプリスト(お気に入りのアドレス)

DQ3_template.png … サンプル画像 兼 アドレスリストの使い方

■使い方
①dq3.*をROMのあるフォルダにコピー
②ROMファイルをdq3.smcにリネーム(.adfや.pal等と名前をあわせる)
③YY-CHRでROM(dq3.smc)を開く
④バンク側のアドレスツールバーの★アイコンからジャンプリストのアドレスへジャンプする




それぞれのファイルを利用すれば、他のゲームでもテンプレートを作る事もできます
　adf/pal/datファイルはYY-CHRで編集可能なバイナリファイルです。
　setting/jumplistファイルはテキストファイルです。データのアドレスを記述します。
